🃏 Texas Hold'em Poker Game
FastAPI와 React를 기반으로 한 전략적 텍사스 홀덤 포커 게임입니다. 
단순한 카드 놀이를 넘어, 딜러와 자산을 두고 경쟁하며 한쪽을 파산시켜야 끝나는 몰입감 있는 게임 경험을 제공합니다.

✨ 주요 기능
💰 자금 대결 및 베팅 시스템
데스매치 구조: 플레이어와 딜러가 각각 $1,000의 자산으로 시작하여 한쪽이 파산할 때까지 대결합니다.

단계별 추가 배팅: 게임 시작(Ante)뿐만 아니라 Flop, Turn, River 단계마다 추가 비용(Call)이 발생하여 실제 포커와 같은 전략적 긴장감을 유도합니다.

전략적 기권(Fold): 패가 좋지 않을 때 빠르게 기권하여 손실을 최소화하는 폴드 메커니즘을 구현했습니다. 기권 시 현재까지 쌓인 판돈은 딜러가 획득합니다.

🏆 정밀한 지능형 로직
실시간 족보 업데이트: 카드가 오픈될 때마다 현재 핸드에서 가질 수 있는 최선의 족보를 실시간으로 계산하여 화면에 표시합니다.

정교한 승패 판정: 동일 족보 발생 시 숫자의 가치(Rank)를 비교하여 승자를 가려내며, 무승부 시 판돈을 공평하게 나누는(Chop) 로직을 포함합니다.

🎨 몰입감 있는 시각 연출
핵심 카드 하이라이트: 쇼다운 시 승리 족보를 완성한 카드들만 골라 금빛으로 강조하여 승리 이유를 직관적으로 보여줍니다.

파산 엔딩 오버레이: 누군가 자금을 모두 잃으면 별도의 브라우저 알림창 대신 화면 전체를 덮는 세련된 결과창(CHAMPION / BANKRUPT)이 나타납니다.

반응형 UI: 상단 자산 표시 바와 하단 컨트롤 버튼 배치를 통해 실제 웹 게임과 같은 조작감을 제공합니다.

🚀 시작하기 (How to Run)
Docker Desktop이 설치되어 있어야 합니다.

프로젝트 루트에서 실행
docker-compose up --build

접속 주소

프론트엔드: http://localhost:5173

백엔드 API: http://localhost:8000

📂 프로젝트 구조
backend/: FastAPI 기반의 게임 엔진 및 자산 관리 엔드포인트

frontend/: React 기반의 인터랙티브 UI 및 실시간 상태 관리

modules/: 덱 생성(Cards), 족보 계산(Evaluator), 승자 판정(Game_Engine) 모듈

💡 앞으로 더 해볼 수 있는 것들 (Next Steps)
사운드 효과: 카드 분배 시의 타격감과 승리 시 칩이 쏟아지는 오디오 연출
