# 🃏 AI-Powered Texas Hold'em Poker Game

FastAPI와 React를 기반으로 한 텍스트 홀덤 포커 게임입니다. 
도커를 통해 어디서든 즉시 실행 가능하며, 부드러운 카드 애니메이션과 실시간 족보 판정 기능을 제공합니다.

## ✨ 주요 기능
- **실시간 족보 계산**: 카드가 추가될 때마다 현재 최선의 족보를 실시간 업데이트
- **순차적 카드 애니메이션**: 카드가 실제 테이블에 놓이는 듯한 리듬감 있는 등장 효과
- **승자 하이라이트**: 쇼다운 시 승리한 플레이어에게 금빛 일렁임 효과 제공
- **도커 지원**: 복잡한 설정 없이 명령어 한 줄로 프론트/백엔드 동시 실행

## 🚀 시작하기 (How to Run)

Docker Desktop이 설치되어 있어야 합니다.

# 프로젝트 루트에서 실행
docker-compose up --build

### 💡 앞으로 더 해볼 수 있는 것들 (Next Steps)

1.  **사운드 효과**: 카드가 깔릴 때 '탁' 소리나, 승리했을 때 '팡파르' 소리를 넣어보세요.
2.  **베팅 시스템**: 지금은 카드만 보지만, 칩(Chip) 개념을 넣어서 베팅 로직을 추가하면 진짜 게임이 됩니다.
3.  **멀티플레이어**: 지금은 딜러와 1:1이지만, WebSockets를 공부해서 실제 다른 유저와 치는 기능을 시도해 볼 수 있습니다.
