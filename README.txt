🃏 Texas Hold'em Poker: Deathmatch Edition

FastAPI와 React 기반의 전략적 텍사스 홀덤 게임입니다. 
단순한 카드 게임을 넘어, 딜러와의 정교한 심리전과 자산 경쟁을 통해 한쪽을 파산시키는 몰입감 있는 게임 경험을 제공합니다.

■ ✨ 주요 기능 및 업데이트 (Update Highlights)

  [💰 정교한 자금 및 베팅 시스템]
  ● 데스매치 구조: 플레이어/딜러 각 $2,000 초기 자산 부여, 한쪽 파산 시 종료
  ● 정산 시퀀스 최적화 (New): 승패 판정 후 [카드 오픈 -> 결과 확인 -> 팟 비우기 -> 자산 업데이트] 순으로 시각적 정산 일치화
  ● 동적 베팅 반영 (New): 딜러의 RAISE/BET 액션 시 즉각적인 데이터 동기화로 실시간 Call 금액 및 UI 업데이트
  ● 전략적 Fold: 패배 예상 시 기권(Fold)을 통한 손실 최소화 연출

  [🏆 지능형 게임 로직 & 턴 제어]
  ● 선공 로직 엄격 적용 (New): 딜러 버튼(D) 위치에 따른 프리플랍/포스트플랍 선공(First Action) 로직 완벽 구현
  ● 메시지 체이닝 (New): 딜러의 이전 페이즈 대응(CALL)과 다음 페이즈 선공 액션(CHECK/RAISE)을 연계하여 끊김 없는 흐름 제공
  ● 라운드 자동 동기화: 유저 액션에 따른 즉각적인 페이즈 전환 및 서버 기반 차기 액션 결정
  ● 실시간 족보 계산: 카드 오픈 단계별 최상의 5장(Best Hand) 실시간 분석 및 하이라이트 제공

  [🎨 비동기 연출 엔진 및 UX/UI]
  ● 시각적 스포일러 방지 (New): 서버 데이터의 즉각 반영을 억제하고 sleep 기반 시퀀스 제어로 실제 포커 흐름 재현
  ● 연출 타이밍 최적화 (New): 각 연출 단계별 지연 시간을 조정하여 긴장감 있는 템포와 쾌적한 조작감 양립
  ● 대화형 인터페이스: 딜러의 모든 의사결정을 실시간 말풍선 메시지로 시각화
  ● 디버깅 추적 시스템 (New): 콘솔 로그를 통한 [TURN], [DATA], [PHASE]별 실시간 로직 트래킹 기능

■ 🚀 시작하기 (Quick Start)

  1. 사전 요구사항: Docker Desktop 설치 필수
  2. 컨테이너 실행: docker-compose up --build
  3. 서비스 접속:
     - Frontend: http://localhost:5173
     - Backend API: http://localhost:8000

■ 📂 프로젝트 구조

  📂 frontend/ : React 기반 UI 및 프론트엔드 비동기 연출 상태 관리 (App.jsx 핵심)
  📂 backend/main.py  : FastAPI 기반 API 서버 환경 설정
  📂 backend/modules/  : 게임 핵심 엔진 및 비즈니스 로직 모듈
    - game_service.py : [핵심] 턴 제어, 페이즈 전환 및 게임 흐름 관리
    - cards.py        : 덱 생성, 셔플 및 카드 분배 로직
    - evaluator.py    : 족보 분석 및 랭킹 계산
    - game_engine.py  : 승패 판정 및 판돈 정산 로직
