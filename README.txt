🃏 AI-Powered Texas Hold'em Poker Game
FastAPI와 React를 기반으로 한 지능형 텍사스 홀덤 포커 게임입니다. 
도커를 통해 즉시 실행 가능하며, 정교한 승패 판정 로직과 몰입감 있는 시각 연출을 제공합니다.

✨ 주요 기능
실시간 족보 계산: 카드 배분 단계마다 현재 핸드에서 가능한 최선의 족보를 실시간 업데이트

정밀한 승패 판정: 동일 족보 상황에서도 카드별 숫자 가치(Power)를 비교하여 정확한 승자 결정

승자 핵심 카드 하이라이트: 쇼다운 시 승리한 핸드의 핵심 카드(2~5장)만 골라 금빛으로 강조

순차적 카드 애니메이션: 실제 테이블 흐름에 맞춘 리듬감 있는 카드 등장 및 플로팅 효과

도커 기반 환경: 명령어 한 줄로 프론트엔드와 백엔드를 즉시 동시 실행 가능

🚀 시작하기 (How to Run)
Docker Desktop이 설치되어 있어야 합니다.

프로젝트 루트에서 실행
docker-compose up --build

프론트엔드: http://localhost:5173

백엔드 API: http://localhost:8000

💡 앞으로 더 해볼 수 있는 것들 (Next Steps)
베팅 시스템: 보유 자산(Bankroll) 관리와 Check, Call, Raise, Fold 로직 추가

사운드 효과: 카드 배분 시 타격음 및 승리 시 팡파르 등 오디오 연출 추가

확률 예측 엔진: 현재 상황을 분석하여 특정 족보가 완성될 확률을 실시간 계산
