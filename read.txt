Pocker

1. backend, frontend 폴더 생성 및 .gitignore 파일 생성

2. backend 경로에 가상환경 설치 python -m venv .venv

3. 가상환경 시작 ./.venv/Scripts/activate

4. pip install fastapi uvicorn 설치
FastAPI: 우리가 짤 파이썬 로직을 웹 API로 만들어주는 도구
Uvicorn: 파이썬 서버를 실제로 구동시켜 브라우저에서 접속 가능하게 해주는 엔진

5. pip install pandas
족보 계산 로직 짤 때 사용할 라이브러리

6. main.py에 코드 작성

7. frontend 경로 npm -v 설치 확인

8. 
# 1. Vite 프로젝트 틀 생성 (설명서 생성)
npm create vite@latest . -- --template react

# 2. 설명서에 적힌 기본 부품(React 등) 모두 설치 (필수!)
npm install

# 3. 추가 부품(Axios) 설치
npm install axios

# 4. 이제 실행!
npm run dev

9. frontend/src/App.jsx, App.css 코드 작성

10. backend/main.py CORS 수정

allow_credentials=True: 쿠키나 인증 헤더 등을 포함한 요청을 허용합니다. 나중에 로그인 기능을 넣을 때 필수적

allow_methods=["*"]: GET, POST, PUT, DELETE 등 모든 방식의 요청을 허용

allow_headers=["*"]: 어떤 형태의 HTTP 헤더가 들어와도 받아주겠다는 뜻

11. 실행
backend : uvicorn main:app --reload
frontend : npm run dev

12. backend/main.py 수정해서 딜러 추가

13. 딜러, 유저, 베이스카드 까지는 나오지만 딜러와 유저의 승패 판별 로직 추가해야함 

14. backend/main.py 모듈화

15. backend/modules 폴더 생성해서 카드, 족보 판정, 승패 판정 로직 추가

16. frontend 파일들도 추가

17. 아직 동일 족보일때 더 높은 숫자가 이기는 로직은 개발하지 못했음
